<프로젝트 주제 : 클라우드 기반 정보수집 시스템>

개발 및 운영 환경 : 도커
사용자 인터페이스 : CLI

파이썬으로 CLI 운영환경 제공 -> 사용자 등록/삭제/권한 할당, 정보 수집 시간, 정보 수집 대상 목록 관리
주기적으로 웹서버 운영 로그 수집 (/var/log/nginx/access_log -> IP 목록 추출, error_log -> Error 코드 등록)
 - 등록된 사용자는  웹서버 운영 로그 조회가 가능해야 함
 - 사용자는 관리자와 일반 사용자로 구분되어야 함 (관리자는 사용자 등록 권한을 가짐)

도커 환경을 위한 docker-compose.yaml 작성
 - 웹서버(관리대상), WAS, CLI 운영환경, DB서버
 - 웹소스, CLI 운영프로그램, DB의 데이터 디렉토리는 volume으로 연결하여 구현

보고서의 개요 부분에 필요한 사항 : 
 - 클라우드 관련 기술 정리
 - 현황 및 향후 전망 (개인적 의견 환영)
